CSS Font-Family Property

Jin Vincent Necesario, Aug 10, 2021

Within CSS, the font-family property is used to specify the font of a text. Otherwise, browsers will display text using its default font. When you assign or specify fonts, it's important to remember that users are unlikely to have the same fonts installed on your computer. If you have defined a font that the user doesn't have, your text will display according to the browser's default fonts, regardless of what you'd prefer.
